Also der Updater macht in der Regel fast alles automatisch.
Er erstellt aus den neuen Addons aus dem Ordner "updated_addons" die Zip Dateien.
Diese kopiert er strukturiert in die jeweiligen Ordner in dem Hauptordner xStream-Repo der gleichzeitig das GIT ist.
ich entferne meine Dateien daraus
.git
.gitignores
.gitmodules
Dies musst du bei dir drin lassen.
Falls dein GIT Ordner anders heist musst du ihn in der generators.py Zeile 15 채ndern


hier eine grobe Auflistung wie der Struktur Aufbau sein sollte
ich habe alle GIT in einem Ordner Names Github du musst es an die Stelle kopieren wo du deine Ordner hast
Bei mir zum Beispiel  D:/Github/Kodi 19

/plugin.video.xstream				# Darin sind die direkten GIT Dateien
/python 							# Darin ist Python
/xStream-Web						# Darin ist die Webseite
/xStream-Repo						# Hier entsteht das neue GIT
/xStream-Backup						# xStream Backup
/updated_addons						# Darei kommen die neuen Addons
/generator.py						# Generator
/imports.py							# Import Datei
/Starte Repo Update.cmd				# Startet das Repo Update
  

Wenn du dann die .cmd startest wird alles in den xStream-Repo Ordner erstellt und du musst dann nur noch nach dem 체berpr체fen ein Commit machen.

falls dein Repo Ordner anders heist 채ndere ihn bitte.
